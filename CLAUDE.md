# SideQuest Bingo

AI-powered bingo board generator that gamifies any event. Players get custom 5×5 bingo boards with squares generated by Claude based on the event type.

## Stack
- **Frontend**: React 19 + Vite (src/)
- **Backend**: Express server (server.js) — proxied at /api during dev
- **AI**: Anthropic Claude Haiku via @anthropic-ai/sdk
- **Styling**: Custom CSS (src/App.css), no UI library

## Running locally
1. Copy `.env.example` to `.env` and add your `ANTHROPIC_API_KEY`
2. `npm install`
3. `npm run dev` — starts Vite (port 5173) + Express (port 3001) concurrently

## Key files
- `server.js` — Express API; POST /api/generate calls Claude for 25 bingo items
- `src/App.jsx` — all state management; renders EventSetup or game view
- `src/components/EventSetup.jsx` — setup form (event name, description, players)
- `src/components/BingoBoard.jsx` — 5×5 grid
- `src/components/BingoSquare.jsx` — individual cell with mark toggle
- `src/components/PlayerTabs.jsx` — player switcher
- `src/components/WinBanner.jsx` — BINGO! win overlay

## Architecture
- All state lives in App.jsx — no Context/Redux needed
- Each player gets the same 25 AI items, independently Fisher-Yates shuffled
- FREE SPACE always lands at board index 12 (center of 5×5)
- Bingo detection: checks 5 rows, 5 cols, 2 diagonals
- Express server loads .env manually via readFileSync (no dotenv dep)
- Vite proxies /api → localhost:3001 in dev; Express serves /dist in production

## AI prompt
The Claude prompt (in server.js) requests 25 short (≤5 words) bingo items specific to the event, with at least 3 action items and one FREE SPACE. Returns a raw JSON array of 25 strings.

## Extending
- Change AI model: update `model` in server.js
- Add persistence: use localStorage in App.jsx for boards/marked state
- Real-time multiplayer: replace local state with Supabase realtime or WebSockets
- More players: raise the 8-player cap in EventSetup.jsx
